Introducing C++

C++ is a cross-platform language that can be used to create high-performance applications.C++ was developed by Bjarne Stroustrup, as an extension to the C language. C++ gives programmers a high level of control over system resources and memory. The language was updated 3 major times in 2011,2014, and 2017 to c++11,C++14,and C++17.

Why Use C++

C++ is one of the world's mot popular programming languages. C++ can be found in today's operating systems, Graphical User Interfaces, and embedded systems. C++ is an Object-oriented programming language which gives a clear stsructure to programs and allows code to be reused, lowering development costs. C++ is fun and easy to learn!!!!!!.
